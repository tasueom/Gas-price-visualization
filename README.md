# 전국 주유소 유가 시각화·분석 시스템

Flask 기반의 주유소 유가 데이터 관리 및 시각화 웹 애플리케이션

## 주요 기능

### 1. 데이터 관리
- **CSV 파일 업로드**: 주유소 유가 정보 일괄 등록
- **데이터 조회**: 페이징, 정렬, 검색 기능 제공
- **결측치 처리**: 0원 데이터를 "판매 X"로 표시

### 2. 데이터 분석 및 시각화
#### 유가 평균 분석 (Chart.js)
- 4개 유종(고급휘발유, 휘발유, 경유, 실내등유)의 평균 가격을 막대 그래프로 시각화
- 검색 조건에 따른 실시간 평균 계산

#### 지점 분석 (Chart.js)
- **브랜드별 지점 수**: 파이 차트로 시장 점유율 표시
- **셀프/일반 비율**: 셀프 주유소와 일반 주유소 분포

#### 유가 분포 분석 (Matplotlib + Seaborn)
- 전국 유가 데이터의 **히스토그램 + KDE** 시각화
- 각 유종별 가격 분포, 평균, 중앙값 표시
- 2×2 서브플롯으로 4개 유종 동시 비교

### 3. 검색 및 필터링
- 고유번호, 상호, 주소, 상표 기준 검색
- 검색 결과에 따른 차트 동적 업데이트

## 기술 스택

### Backend
- **Flask**: 웹 프레임워크
- **MySQL**: 데이터베이스
- **Pandas**: 데이터 처리
- **Matplotlib + Seaborn**: 통계 그래프 생성

### Frontend
- **Chart.js**: 인터랙티브 차트

## 실행 방법

1. 필요한 패키지 설치
```bash
pip install flask pandas mysql-connector-python matplotlib seaborn
```

2. 데이터베이스 설정 (db.py 수정)
```python
# MySQL 연결 정보 입력
host, user, password, database
```

3. 애플리케이션 실행
```bash
python app.py
```

4. 브라우저에서 접속
```
http://localhost:5000
```

## 데이터 형식

CSV 파일은 다음 컬럼을 포함해야 합니다:
- 고유번호, 지역, 상호, 주소, 상표, 셀프여부
- 고급휘발유, 휘발유, 경유, 실내등유 (가격, 단위: 원)
- CSV 업로드 페이지에 있는 링크에서 다운받을 수 있습니다.
